## Option B — Detailed (4 bullets, good for a full project section)

**Shopping Behaviour Analytics Dashboard with Machine Learning**
*Python · Pandas · Scikit-learn · Power BI · Streamlit · Data Visualization*

- Designed and built a full-stack business analytics project analyzing customer shopping behaviour
  across 3,900+ transactions, covering data cleaning, feature engineering, and exploratory data
  analysis using Python (Pandas, NumPy, Matplotlib, Seaborn).
- Engineered a "High Value Customer" classification target and trained a Decision Tree Classifier in
  Scikit-learn, achieving 76% test accuracy; evaluated model performance with confusion matrix,
  precision/recall/F1 metrics, and feature importance analysis.
- Developed a 3-page interactive Power BI dashboard (Executive Overview, Customer Behaviour Analysis,
  ML Prediction) with KPI cards, cross-filtering slicers, heatmaps, and DAX measures translating raw
  transaction data into business-ready insights.
- Built a companion Streamlit application for live, single-customer ML predictions and structured the
  project into a professional, GitHub-ready repository with full documentation for portfolio and
  interview presentation.
